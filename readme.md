# Monitor de Señales TCP/IP en Tiempo Real

## Requisitos previos
- Python 3.7 o superior
- Entorno virtual (recomendado)


## Configuración inicial

1. **Activar el entorno virtual**:
   ```bash
   # Windows
   venv\Scripts\activate

   # Linux/MacOS
   source venv/bin/activate

2. **Instalar depencias**:

    pip install -r requirements.txt
 

# Codigo para ejecutar el programa

    Para que se pueda funcionar todo el programa , es necesario
    activar el entorno virtual, en la raiz del proyecto

1. **Activar entorno virtual**:

    ```bash
    venv\Scripts\activate
    ```

2. **Activar [TCP-CLIENT] archivo principal raiz del proyecto**:

    python main.py

3. **Backend (Python):** carpeta `WebSocket/protocols/`, archivo `tcp_server_tiempo.py`


    python tcp_server_tiempo.py
